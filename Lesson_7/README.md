there are two type of array methods
    Mutable:
        push() 
            adds one item to the last part of the array
        pop()
            removes one item from the last part of the array
        unshift()
            add one item to the beginning part of the array
        shift()
            removes one item from beginning part of the array
        splice()
            it remove the items from array and puts new item
    Immutable:
        sort()
            sort the array in ascending order
        reverse()
            reverse the array 
        include()
            shows if one items includes or not
        indexOf()
            shows the index of the item
        toString()
            converts the array to string 
        join()
            adds array elements to one string
        slice()
            it slice the array it returns a new list 
        concat()
            add two arrays to one array
