Operators
    ==
        compares only values
    ===
        compares both values and types
    !=
        compares only values
    !==
        compares both values and types
    >=
        greater than or equal to
    <=
        less than or equal to
    >   
        greater than
    <
        less than
    &&
        same with AND operator
    ||
        same with OR operator
    !
        return true for falsy values
    !!
        return true for truthy values    
    Ternary operator
        short and universal version of if statement
    Nullish operator
        returns a given value if a variable is null
